New to Opera

Familiarize yourself with Opera by browsing the help files. You can read about how Opera handles tabs, find out how to use the integrated searches, and learn how to set up Opera to your liking using the preferences.

New in Opera 10.50

Opera 10.50 introduces private tabs, a new main menu, and Opera Widgets, a technology that frees widgets from the browser user interface, and gives them the status of regular applications.
