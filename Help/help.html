Downloads

Wherever you see the PDF-XChange logo you can download a booklet in Adobe’s Portable Document Format. If you want to download several books, click on the icon above for a complete list of available downloads.

Wherever you see the Calibre logo you can download an E-book in epub format for reading on a portable device or in the Calibre viewer.

Booklet Printing

Most of my booklets are now formatted for the convenience of reading online. If you want a printed hard copy, it is not difficult to print any PDF file as a booklet using PDF-XChange Viewer.

Website Installation

Downloaded the website from the home page as AIM Website.exe or AIM Website.zip. Extract the files to any convenient folder. Open \AIM Website\index.html to browse off-line. MP3 and PDF content is not available for offline use.

Website Background

If you don’t like the background texture it is easy to change it if you have downloaded the website to your hard disk. Replace the file AIM_Randomf.jpg with your own or just delete it to get a plain white background.

Pāli Pronunciation Guide

ā in Pāli is pronounced as a in Bali
ī in Sīla is pronounced as ee in feeler
ū in Rūpa s pronounced as u in super
