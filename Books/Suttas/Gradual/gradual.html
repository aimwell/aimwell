Anguttaranikāya

The Gradual Sayings

Translated by Bhikkhu Pesala

This early edition is still work in progress. Although it is neither complete, nor very polished, my view is that even a poor translation is better than none at all. Since it is in parallel with the Pāli text, students of Pāli will be able to compare it with the original text and to other translations.

My aim was to provide a convenient reference for internet discussion. For that purpose each chapter and verse has an HTML anchor so that one can link directly to that point on the page. The naming scheme is, I hope, intuitive. To link to verse 298 in the Book of Ones, just append “#298” to the page address — http://www.aimwell.org/Books/Suttas/Gradual/Ones/ones.html#298 or to link to the chapter named ”Mindfulness of the Body” append “#MindfulnessoftheBody” to the page address. Where two or more verses are grouped together, only the first verse has an HTML anchor name.
